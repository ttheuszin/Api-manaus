# API Manaus 🌴

Primeira API REST feita 100% pelo celular usando Termux.

## 🚀 Live Demo
**API:** https://api-manaus-1.onrender.com

## 📍 Endpoints
| Rota | Descrição |
| --- | --- |
| `/` | Status da API |
| `/hora` | Hora atual de Manaus |
| `/cep/<numero>` | Busca endereço via ViaCEP |
| `/clima` | Temperatura atual via Open-Meteo |

## 🛠️ Stack
Python + Flask + Render + Git/GitHub

Feito em 4 dias, direto de Manaus-AM.
